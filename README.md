# Melina

Uma ferramenta simples e eficiente para baixar vídeos do YouTube em formato MP3 (somente áudio) ou MP4 (áudio e vídeo). Utilizando uma interface gráfica amigável desenvolvida com Flet, Melina facilita o gerenciamento de links de download e a seleção do formato desejado.

É necessário instalar o ffmpeg:

No Windows: 
- winget install "FFmpeg (Essentials Build)"

No Linux:
- sudo apt install ffmpeg
